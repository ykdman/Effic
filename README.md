# Efficieneer Project

Hope to provide you with efficiency.

## Project Description

- 효율적인 일정 관리, 및 하루 기록을 위한 서비스
- Note : 하루 기록 및, 메모 기능을 이용한 텍스트 기록 서비스
- ToDo : 할 일 목록 작성 서비스
- Calender : ToDo와 연결 되어, 일정별 할 일 확인 서비스
